# DCF 估值報告：Apple Inc.

## 綜述
貼現現金流（DCF）估值是估算企業內在價值的黃金標準。這種方法回答了一個根本問題：**「基於未來可能產生的現金流，這家企業的價值是多少？」** 不同於相對估值，DCF能夠告訴您該業務在絕對意義上值多少，獨立於市場情緒或同業定價。

DCF 要求嚴謹的假設，較小的增長率、利潤率或折現率假設上的變化會在10年內顯著增長。因此，應模擬三種情境（樂觀情境、基礎情境、悲觀情境），並提供敏感性表，以免假設顯得過於絕對。

## DCF 方法論步驟

### 步驟1：建立基礎指標

- **過去十二個月（TTM）自由現金流**：營運現金流減去資本支出
- **營收基期**（TTM 營收及最近一個完整財年）
- **自由現金流利潤率**（TTM FCF / TTM Revenue × 100%）
- **流通股數量**（稀釋後 — 包括未歸屬股票報酬和期權進行SBC調整）
- **淨負債**：總債務減去現金及等價物（負淨負債意味著淨現金狀況）
- **股票報酬計劃（SBC）** — 作為實際稀釋成本識別；若未排除，從自由現金流中扣去
- **有效稅率**（以TTM為基礎，正常化一次性稅務事件）

### 步驟2：營收增長預測（1–10年）

使用多種錨點來校準可信的增長假設：

- **分部法**：若可能，單獨預測各個營收分部
- **歷史增長分析**：3年、5年和10年營收複合年增長率作為基準錨
- **分析師一致預測**：使用賣方共識的前3年作為交叉檢查
- **管理層指引**：前瞻性營收指引和長期目標
- **行業增長率**：視其為最高錨
- **增長遞減**：在1-5年內應用較高增長，6-10年遞減至終端增長率
- **三個情境**：樂觀（市場份額增長，利潤率擴張）、基礎（歷史趨勢）、悲觀（競爭壓力，宏觀逆風）

### 步驟3：自由現金流利潤率假設

基於運營槓桿和商業模式動態預測未來自由現金流利潤率：

- **歷史自由現金流利潤率趨勢**（擴展、穩定或壓縮）
- **運營槓桿潛力**：當營收擴大時，哪些固定成本可被利用？
- **資本支出強度**（佔營收百分比）：是否在增加或減少？
- **營運資金變化**：是消耗者還是生成者？
- **正常化一次性項目**：剝離訴訟和解金、結構重整費用
- **SBC調整**：從報告的營運現金流中扣除SBC以得到真實經濟自由現金流

### 步驟4：終端價值計算

終端價值代表第10年以後所有現金流：

- **終端增長率 (g)**：通常為2-3%（以名義GDP增長為錨）。絕不要設定g大於WACC，這意味著無限價值
- **戈登增長模型（首選）**：
  ```
  TV = FCF₁₀ × (1 + g) / (WACC − g)
  ```
- **退出倍數法（替代方法）**：
  ```
  TV = FCFₙ × (EV / FCF exit multiple)
  ```
- **終端價值佔企業價值百分比**：如果TV超過總EV的80%，明確表示並擴大敏感性範圍

### 步驟5：加權平均資本成本（WACC）計算

**股權成本（CAPM）**：
```
Ke = Rf + β × (Rm − Rf)
```
- **Rf** = 無風險利率（當前10年期美國國債收益率）
- **β** = 股票β值（5年每月）
- **Rm − Rf** = 股票風險溢價（使用Damodaran當前的ERP，歷史上為5-6%）
- **小市值溢價**：對於中小市值公司加0.5–2%

**債務成本**：
```
Kd = (Interest Expense / Total Debt) × (1 − Effective Tax Rate)
```

**資本結構權重**（使用市值權重，而非賬面價值）：
- **E/V** = 市值 / (市值 + 總債務)
- **D/V** = 總債務 / (市值 + 總債務)

**WACC公式**：
```
WACC = Ke × (E/V) + Kd × (D/V)
```

**典型的WACC範圍按風險等級分類**：
```
風險等級            WACC範圍    公司示例
───────────────────────────────────
低風險（公用事業）   6–8%       法規受控公用事業、大盤股必需品
中風險              8–11%      大盤科技、成熟增長型
高風險              11–15%     小盤、發展中市場、週期性
非常高的風險         15–20%+    早期階段、困境中、無收入
```

### 步驟6：折現現金流

```

終端價值現值 = TV / (1 + WACC)¹⁰

企業價值 = Σ 所有現金流現值（1至10年）+ 終端現值

股權價值 = 企業價值 − 淨負債
  （如果公司有淨現金記錄，則加回淨現金）

內在價值每股 = 股權價值 / 稀釋後股份
```

## 常見的DCF陷阱

1. **垃圾進，垃圾出**：將最近的高增長率過度外推到未來。尤其在第6至10年，保持謹慎。
2. **終端價值主導**：如果TV超過企業價值的70%，請積極對終端假設進行壓力測試。
3. **WACC過低**：忽略小市值/流動性溢價之公司；此方法會虛增內在價值。
4. **忽視周期性**：對於周期性企業，使用峰值自由現金流利潤率。始終正常化並持續通過整個業務週期。
5. **忽略股票報酬計劃**：SBC是實際、稀釋的成本。從營運現金流中扣除。
6. **單情境思維**：始終運行樂觀、基礎和悲觀場景。
7. **隱藏的營運資金和資本支出變化**：快速增長的公司通常需要消耗大量營運資金。
8. **貨幣與地理混合**：對於國際企業，需要依地理區域預測並使用適當的折現率。

## DCF vs. 相對估值

**當DCF最具可靠性時**：
- 穩定、成熟的業務，具有可預測且持續一致的自由現金流
- 資產負債輕的企業，自由現金流轉換率高（軟件、消費品牌）
- 擁有10年以上自由現金流生成歷史的公司

**當DCF較不可靠時**：
- 早期階段增長公司，無正自由現金流
- 高週期性企業，正常化自由現金流需要重大判斷
- 金融公司（銀行、保險、房地產投資信託） — 使用市淨率、市盈率或股息折現模型
- 調整情況，盈利路徑不確定時
