## 步驟 5：WACC 計算

### 自有資金成本 (CAPM)
```
Ke = Rf + β × (Rm − Rf)
```

- *Rf = 無風險利率*: 當前10年期美債收益率
- *β = 股票貝塔系數*: 過去5年每月
- *Rm − Rf = 股權風險溢價*: 使用Damodaran的當前ERP，歷史上為5–6%
- *規模溢價*: 為中小型公司加0.5–2%

### 債務成本
```
Kd = (利息支出 / 總債務) × (1 − 有效稅率)
```

### 資本結構權重

- *E/V = 市值 / (市值 + 總債務)*
- *D/V = 總債務 / (市值 + 總債務)*

### WACC 公式
```
WACC = Ke × (E/V) + Kd × (D/V)
```

## 步驟 6：現金流折現

折現每年度的現金流，然後計算出企業價值：

```
企業價值 = Σ 折現現金流 (年1–10) + 折現終端價值

股權價值 = 企業價值 − 淨債務

每股內在價值 = 股權價值 / 稀釋後股數
```

## 步驟 8：安全邊際評估

```
安全邊際 = (內在價值 − 市場價格) / 內在價值 × 100%

上行潛力 = (內在價值 − 市場價格) / 市場價格 × 100%
```

**評估等級**：

## 常見DCF陷阱

1. **垃圾進，垃圾出**: 不宜過度延長高增長率至未來。尤其是第6–10年要保持保守。
2. **終端價值占比過高**: 若TV > 70%企業價值，請嚴格測試終端假設。
3. **WACC太低**: 忽視小型公司的規模/流動性溢價會人為地增大內在價值。
4. **忽視周期性**: 使用周期性業務的峰值FCF邊際。永遠應正常化整個商業周期的FCF。
5. **忽視股份補償成本**: SBC是一種真實的稀釋成本。從營業現金流中扣除。
6. **單一情景思考**: 必須運行牛、市和熊三種情景。
7. **隱藏的營運資金和資本支出變化**: 快速增長的公司通常消耗大量營運資金。
8. **貨幣和地理組合**: 對於國際企業，按區域進行預測並使用適當的折現率。

## DCF與相對估值比較

**當DCF最為可靠**：
- 穩定、成熟的企業且有可預測的一致性FCF
- 資產輕型企業，高FCF轉化率（如軟件、消費品牌）
- 擁有10年以上FCF生成歷史的公司

**當DCF不太可靠**：
- 無正FCF的初期增長公司
- 高度周期性企業，需顯著判斷以正常化FCF
- 金融公司（銀行、保險、REIT）：應使用市價/市值、市價/收益或股息折現模型
- 業務反轉情況，不確定盈餘改善路徑
