# 蘋果公司 (Apple Inc.) DCF 評價報告

## 概述

折現現金流量法 (DCF) 是評估內在價值的黃金標準。此方法旨在回答核心問題：**"根據未來產生的現金流，這家企業的價值是多少？"** DCF 評價與市場情緒或同業定價無關，給出的是企業的絕對價值。

在 DCF 的建模中，需要嚴謹的假設。由於在 10 年的時間範圍內，小的變更在成長率、利潤率或折現率假設中會產生重大的複合影響。永遠模擬三種情景（樂觀/基礎/悲觀），並提供敏感性表格，確保假設不是單一估計。

## 步驟化DCF方法

### 步驟1：建立基礎指標

在展望未來之前，收集並記錄當前的基礎資料：

- **近十二個月 (TTM) 自由現金流量** = 營運現金流量 − 資本支出
- **收入基礎**（TTM 收入和最近一個完整財年的收入）
- **自由現金流量率** (FCF margin) = TTM FCF / TTM 收入 × 100%
- **流通股數**（攤薄後 — 包括未授予的 RSUs 和選擇權進行股權補償調整）
- **淨負債** = 總負債 − 現金及等價物（負債為負值表示淨現金狀況）
- **股權補償 (SBC)** — 確認為真正的攤薄成本；若 FCF 未扣除，應從營運現金流中扣除
- **實效稅率** (TTM 基礎，需排除一次性稅務事件的影響)

### 步驟2：收入增長預測（第1至第10年）

利用多個依據推斷合理的成長假設：

- **分部法**：可能的情況下，分別預測每個收入分部
- **歷史增長分析**：以 3年、5年和10年收入年均增長率 (CAGR) 為基準依據
- **分析師一致性預測**：使用分析師共識預測第1到第3年的數據以作為交叉檢查
- **管理層指引**：包括前瞻性收入指引和長期目標
- **行業增長率**：作為上限的參考依據
- **增長逐步減慢**：從第1至第5年使用較高的增長率，第6至第10年逐步減到終端增長率
- **三種情景**：樂觀（市場份額增加、盈利率提升）、基礎（歷史趨勢）、悲觀（競爭壓力、宏觀因素不利）

### 步驟3：自由現金流量率假設

根據營運槓桿和商業模式預測未來自由現金流量率：

- **歷史 FCF 率趨勢**（擴大、穩定或縮小）
- **營運槓桿效應**：當收入擴展時，哪些固定成本正在被利用？
- **資本密集度**（占收入百分比）：增加或減少？
- **營運資金變化**：是消耗者還是創造者？
- **正常化一次性項目**：剔除訴訟賠償、重組費用
- **SBC 調整**：從報告的營運現金流量中扣除 SBC，以獲得真實的經濟自由現金流量

### 步驟4：終端價值計算

終端價值代表第十年以後所有的現金流量：

- **終端成長率 (g)**：一般為2–3%（與名義 GDP 增長對應）。g 不能大於 WACC，否則意味著無限價值
- **戈登增長模型（優先）**：
  ```
  TV = FCF₁₀ × (1 + g) / (WACC − g)
  ```
- **退出倍數法（備選）**：
  ```
  TV = FCFₙ × (EV / FCF exit multiple)
  ```
- **終端值占企業價值的比例**：若 TV > 80% 的總 EV，應明確註記並擴大敏感性範圍

### 步驟5：加權平均資本成本 (WACC) 計算

**股本成本 (CAPM)**：
```
Ke = Rf + β × (Rm − Rf)
```
- Rf = 無風險利率（當前10年美國國債收益率）
- β = 股票β（5年按月計算）
- Rm − Rf = 股權風險溢價（使用 Damodaran當前 ERP，歷史上5–6%）
- 規模溢價：對於小/中型公司加0.5–2%

**債務成本**：
```
Kd = (利息支出 / 總負債) × (1 − 實效稅率)
```

**資本結構權重**（使用市場價值權重，而非賬面價值）：
- E/V = 股本市值 / (股本市值 + 總負債)
- D/V = 總負債 / (股本市值 + 總負債)

**WACC 公式**：
```
WACC = Ke × (E/V) + Kd × (D/V)
```

**按照風險水平劃分的典型 WACC 範圍**：
```
風險水平         WACC 範圍    公司示例
────────────────────────────────────────────
低風險 (公用事業)  6–8%          受監管的公用事業、大型基石商品
中等風險          8–11%         大型科技股，已建立的成長公司
高風險            11–15%        小型股、新興市場、周期性公司
非常高風險        15–20%+       早期階段、困境公司、未上市前
```

### 步驟6：現金流折現

```

終端值現值 = TV / (1 + WACC)¹⁰

企業價值 = Σ PV(FCF 年份 1–10) + PV(終端值)

股本價值 = 企業價值 − 淨負債
  (如果公司淨現金狀態，需加回淨現金)

每股內在價值 = 股本價值 / 攤薄流通股數
```

### 步驟8：安全邊際評估

```
安全邊際 = (內在價值 − 市場價格) / 內在價值 × 100%

上升潛力 = (內在價值 − 市場價格) / 市場價格 × 100%
```

**評估標尺**：

## DCF 評估常見陷阱

1. **垃圾進，垃圾出**：將近期高增長率推算至未來太遠。尤其在第6至10年要保持保守。
2. **終端價值主導**：如果 TV > 70%的企業價值，需積極對終端假設進行壓力測試。
3. **WACC 太低**：忽視小型公司的規模/流動性溢價，會人為提高內在價值。
4. **忽視周期性**：在周期性公司中使用峰值 FCF 率。總是應通過整個商業周期來標準化 FCF。
5. **忽略股權補償費用**：SBC 是真實的攤薄成本。需自營運現金流中扣除。
6. **只考慮單一情境**：始終執行樂觀、基礎和悲觀情境。
7. **隱藏的營運資金和資本支出變動**：快速成長的公司通常消耗大量營運資金。
8. **貨幣和地理混合**：對於國際企業，應按地理位置進行預測，並使用適當的折現率。

## DCF 與相對估值比較

**當 DCF 最可靠時**：
- 穩定、成熟的企業具有可預測、穩定的現金流量
- 資產輕型的企業具有較高的 FCF 轉換率（例如軟體、消費品牌企業）
- 擁有 10 年以上現金流生成歷史的公司

**當 DCF 不太可靠時**：
- 早期階段的增長型企業尚未有正的 FCF
- 高周期性企業，其中標準化 FCF 需要大量判斷
- 金融公司（銀行、保險、REITs）— 應使用市淨率、市盈率或股利折現模型
- 輪回型企業，其中盈利能力的路徑不確定
