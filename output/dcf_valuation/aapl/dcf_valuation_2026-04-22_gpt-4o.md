# 蘋果公司 (AAPL) 折現現金流量 (DCF) 價值評估報告

## 概述

折現現金流量模型 (DCF) 是評估公司內在價值的黃金標準。它專注於未來現金流量的預測，當前反映市場情緒的估值相對不確定，DCF 提供了評估商業整體價值的絕對手段。要注意的是，DCF 模型需要謹慎的假設，因為在較長的預測期內，小幅的增長率、利潤率或貼現率的改變都會對估值造成顯著的影響。本報告將以三種情境 (樂觀、基準、悲觀) 提供不同假設下的價值預測。

### 第四步：終端價值計算

終端價值代表超過10年預測期的所有現金流量：

- **終端增長率 (g)**：通常設為2–3%（以名義GDP增長為基準）。切勿設g > 貨幣平均資本成本(WACC)。

### 第五步：加權平均資本成本 (WACC) 計算

**股權成本(CAPM)**:
```
Ke = Rf + β × (Rm − Rf)
```
- Rf = 無風險利率（目前10年期美國國債收益率）
- β = 股票beta值（5年每月）
- Rm − Rf = 股權風險溢價（使用Damodaran當前ERP，歷史上5–6%）
- 小型企業溢價：為小型/中型公司增加0.5–2%

**負債成本**:
```
Kd = (利息支出 / 總債務) × (1 − 有效稅率)
```

**資本結構權重**（使用市場價值而非賬面價值）：
- E/V = 市值 / (市值 + 總負債)
- D/V = 總負債 / (市值 + 總負債)

**加權平均資本成本公式（WACC）**：
```
WACC = Ke × (E/V) + Kd × (D/V)
```

**風險輪廓典型WACC範圍**：

## DCF常見誤區

1. **垃圾進，垃圾出**：過度外推最近高增速。尤其在第6–10年要保守。
2. **終端價值主導**：如TV > 70% 企業價值，需對終端假設進行壓力測試。
3. **WACC過低**：忽視小型公司流動性溢價會虛增內在價值。
4. **忽視周期性**：對周期性業務使用峰值自由現金流利潤率。務必常態化現金流。
5. **忽視股權補償**：股權補償是實際的稀釋成本。從營運現金流中減去。
6. **單一情境思考**：務必運行樂觀、基準、悲觀情境。
7. **隱性營運資金和資本支出變化**：快速增長企業通常需要大量營運資金。
8. **貨幣和地域組合**：對跨國公司需按地區進行預測並採用適合的折現率。

## DCF對比相對估值

**當DCF最可靠**：
- 穩定、成熟的公司，現金流穩定可預測
- 資產輕型企業，高現金轉換率（如軟體、消費品牌）
- 有10年以上自由現金流生成歷史的公司

**當DCF不太可靠**：
- 早期成長公司，未有正現金流
- 高度周期性企業，需顯著判斷才能常態化現金流
- 金融公司 (如銀行、保險、REITs) — 建議使用市淨率、市盈率或股息折現模型
- 改革階段的公司，盈利路徑不明
