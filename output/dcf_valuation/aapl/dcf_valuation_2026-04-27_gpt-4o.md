# 蘋果公司（AAPL）DCF估值報告

## 概述

現金流貼現模型（DCF）是評估企業內在價值的黃金標準。此方法尋求回答一個根本問題：「基於該企業生成的未來現金流，這個企業的真正價值為何？」DCFs不受市場情緒或同儕定價影響，而是獨立來 確定企業的絕對價值。

進行DCF分析時，需要謹慎設置假設。增長率、利潤率或折現率等假設的細微變化會在10年內產生巨大影響。因此，模型應該包括三種情景（牛市/基準/熊市）和敏感性分析。

## DCF方法論步驟

### 第1步：建立基礎指標

蒐集並記錄當前基準數據，以便向前預測：

- **過去十二個月（TTM）自由現金流** = 營運現金流 − 資本支出
- **營收基礎**（TTM營收和最新一財年數據）
- **自由現金流利潤率**（TTM現金流 / TTM營收 × 100%）
- **流通股數**（稀釋後，包括未配股RSU和期權作為股權報酬調整）
- **淨負債** = 總負債 − 現金及現金等價物（負淨負債=淨現金狀態）
- **股權報酬計劃（SBC）** — 確認為實際攤薄成本；若未排除，從FCF中扣除
- **實際稅率**（TTM基準，需正常化以消除一次性稅務事件）

### 第2步：營收增長預測（第1年至第10年）

使用多個支點確認可信的增長假設：

- **分段法**：若可能，分別預測各營收分部
- **歷史增長分析**：3年、5年與10年營收年均增長率作為基準錨點
- **分析師共識預測**：可用賣方共識作為第1至3年交叉檢查
- **管理層指引**：前向指引和長期目標
- **行業增速**：用作上限錨點
- **增長減緩**：在第1至5年使用高增長，可在第6至10年減速向終端增長速率靠近
- **三種情景**：牛市（市場份額獲取，利潤率擴大）、基準（歷史趨勢）、熊市（競爭壓力，宏觀逆風）

### 第3步：自由現金流利潤率假設

基於操作槓桿和業務模式動態，預測未來自由現金流利潤率：

- **歷史自由現金流利潤率趨勢**（擴大、穩定或壓縮）
- **操作槓桿潛力**：隨著收入增長，哪些固定成本得到槓桿利用？
- **資本支出密集度**（佔營收百分比）：增加還是減少？
- **營運資本變化**：消耗還是生成現金？
- **正常化一次性項目**：剔除訴訟和重組費用
- **SBC調整**：從報告的現金流量中扣除SBC，以獲得真實的經濟自由現金流

### 第4步：終端價值計算

終端價值代表超過10年明確預測期間所有的現金流：

- **終端增長率（g）**：通常為2-3%（錨定於名義GDP增長），g勿大於WACC—這將暗示無限價值
- **戈登增長模型（優先法）**：
  ```
  TV = FCF₁₀ × (1 + g) / (WACC − g)
  ```
- **退出倍數法（替代法）**：
  ```
  TV = FCFₙ × (EV / FCF退出倍數)
  ```
- **終端值作為企業價值的百分比**：若TV超過企業總價值80%，需明確標註並增加敏感性範圍

### 第5步：加權平均資本成本（WACC）計算

**股本成本（CAPM）**：
```
Ke = Rf + β × (Rm − Rf)
```
- Rf = 無風險利率（目前10年期美國國債收益率）
- β = 股票貝他（5年每月）
- Rm − Rf = 股權風險溢價（使用Damodaran當前ERP，歷來為5-6%）
- 大小溢價：小型/中型公司增加0.5%-2%

**債務成本**：
```
Kd = (利息支出 / 總負債) × (1 − 實際稅率)
```

**資本結構權重**（使用市值權重而非賬面值）：
- E/V = 股本市值 / （股本市值 + 總負債）
- D/V = 總負債 / （股本市值 + 總負債）

**WACC公式**：
```
WACC = Ke × (E/V) + Kd × (D/V)
```

### 第6步：現金流折現

```
企業價值 = Σ 折現（FCF年1-10） + 折現（終端價值）

股權價值 = 企業價值 - 淨負債

內在價值每股 = 股權價值 / 稀釋流通股
```

### 第8步：安全邊際評估

```
安全邊際 = (內在價值 − 市場價格) / 內在價值 × 100%

上行潛力 = (內在價值 − 市場價格) / 市場價格 × 100%
```


## 常見DCF陷阱

1. **垃圾進，垃圾出**：將近期高增長率推進過遠的未來。對第6至10年尤其要保守。
2. **終端價值主導**：如果TV佔企業價值超過70%，應積極測試終端假設。
3. **低估的WACC**：忽視小型公司規模/流動性溢價會錯誤抬高內在價值。
4. **忽視周期性**：使用周期性業務的峰值自由現金流利潤率。始終正常化自由現金流以涵蓋完整商業周期。
5. **忽視股權報酬計劃**：SBC是實際的攤薄成本。從營運現金流中扣除。
6. **單一情景思維**：始終運行牛市、基準和熊市情景。
7. **隱藏的營運資本和資本支出變化**：快速增長公司通常消耗大量營運資本。
8. **貨幣和地理混合**：對於跨國企業，按照地理位置進行預測並使用相應的折現率。

## DCF與相對估值

**DCFs最具可靠性時**：
- 穩定、成熟的企業，具備可預測、一致的自由現金流
- 資產輕型企業，具有高自由現金流轉換率（軟件、消費品品牌）
- 擁有10年以上自由現金流量生成歷史的公司

**DCFs不太可靠時**：
- 初期增長公司，無正自由現金流
- 高度周期性的企業，需要大量主觀判斷來正常化自由現金流
- 金融公司（銀行、保險、房地產投資信託）——採用市淨率、市盈率或股利折現模型
- 轉型中的企業，盈利之路不明確
