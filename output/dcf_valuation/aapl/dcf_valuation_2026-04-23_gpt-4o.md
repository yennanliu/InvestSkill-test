**股權成本 (CAPM)**：

**債務成本**：
```

```

**資本結構權重**：
- E/V = 市值 / (市值 + 總債務)
- D/V = 總債務 / (市值 + 總債務)

**WACC 計算**：
```
WACC = Ke × (E/V) + Kd × (D/V)
```

## 常見 DCF 錯誤風險點

1. **數據質量不足**：不應以近期高增長率來預測過長的未來。
2. **終端價值過高**：TV > 70% 需謹慎檢驗終端假設。
3. **低估 WACC**：忽略小型公司和流動性溢價。
4. **無視周期性**：周周期性強的行業 FCF 需正常化。
5. **忽略 SBC**：應扣除股權薪酬。
6. **僅單一情境思考**：牛市、基本、熊市場景需多方考量。
7. **工作資本及資本支出變動隱藏**。
8. **貨幣和地理混合**：跨國業務需具有地理折現率預測。
