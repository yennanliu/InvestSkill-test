# 蘋果公司（AAPL）DCF 評估報告

## 概述

DCF 模型是一種估計公司內在價值的重要方法，通過預測未來現金流量並將其折現到當前估算公司價值，它獨立於市場情緒和同業定價。我們通過三個情境（樂觀、基本、悲觀）及敏感性表格展示假設的影響。

## DCF 評估方法

### 第一步：確定基準數據

收集目前的基線數據以進行未來的預測：

- **追蹤十二個月（TTM）自由現金流** = 營運現金流 - 資本支出
- **收入基礎**（TTM 收入與最近完整財年）
- **自由現金流利潤率（FCF Margin）** = （TTM FCF / TTM Revenue）× 100%
- **已發行股票數量（稀釋後）**（包括未授予的RSUs和SBC調整的選擇權）
- **淨負債** = 總債務 - 現金及等價物（負淨負債 = 淨現金狀況）
- **基礎報酬**—作為現實的稀釋成本；若未剔除則從 FCF 中扣除
- **有效稅率**（TTM基礎，排除一次性稅務事件）

### 第二步：營收增長預測（第1至10年）

利用多個參考點推導出合理的增長假設：

- **分段法**：分別預測各收入分部
- **歷史增長分析**：3年、5年和10年收入CAGR作為基線參考
- **分析師共識預測**：使用賣方共識預測
- **管理層指引**：收入指導和長期目標
- **行業增長率**：作為天花板參考
- **增長趨緩**：前5年高增長，第6至10年增速放緩直至終端增長率
- **三個情境**：樂觀（市場份額增加、利潤率擴張）、基本（歷史趨勢）、悲觀（競爭壓力、宏觀經濟逆風）

### 第三步：FCF 利潤率假設

根據經營槓桿及商業模式動態預測未來FCF 利潤率：

- **歷史FCF利潤率趨勢**（擴大、穩定或收縮）
- **經營槓桿潛力**：隨著收入擴張，哪些固定成本受到槓桿影響？
- **資本支出強度**（收入百分比）：增加或減少？
- **營運資金變動**：消耗還是生成？
- **一次性項目正常化**：排除訴訟賠償、重組支出
- **SBC 調整**：從報告的營運現金流中扣除 SBC 以獲得真實的經濟 FCF

### 第四步：終端價值計算

終端價值代表超出10年明確預測期的所有現金流量：

- **終端增長率（g）**：通常為 2–3%（錨定至名義 GDP 增長）。永遠不要設 g > WACC — 會暗示無限價值
- **戈登增長模型（優化）**：
  ```
  TV = FCF₁₀ × (1 + g) / (WACC − g)
  ```
- **退出倍數法（替代）**：
  ```
  TV = FCFₙ × (EV / FCF 退出倍數)
  ```
- **終端價值作為企業價值百分比**：若TV > 80% 全部 EV，需明確標示並擴大敏感性範圍

### 第五步：WACC 計算

**股權成本（CAPM）**：
```
Ke = Rf + β × (Rm − Rf)
```
- Rf = 無風險利率（當前 10 年期美國國債收益率）
- β = 股票 Beta（5 年每月）
- Rm − Rf = 股權風險溢價（使用 Damodaran 當前 ERP，歷史為 5–6%）
- 規模溢價：小型/中型公司的規模溢價增加 0.5–2%

**債務成本**：
```
Kd = (利息支出 / 總債務) × (1 − 有效稅率)
```

**資本結構權重**（使用市場價值權重而非賬面價值）：
- E/V = Equity Market Cap / (Equity Market Cap + Total Debt)
- D/V = Total Debt / (Equity Market Cap + Total Debt)

**WACC 公式**：
```
WACC = Ke × (E/V) + Kd × (D/V)
```

**根據風險特徵的典型 WACC 範圍**：
```

─────────────────────────────────────────────────────

中等風險            8–11%       大型科技、成熟的成長型企業
高風險              11–15%      小型股、新興市場、周期性

```

## 常見 DCF 誤區

1. **垃圾進，垃圾出**：近期高增長率過度外推至未來。特別是在第6–10年時保守。
2. **終端價值主導**：若TV > 70% 企業價值，需積極測試終端假設。
3. **WACC 過低**：忽視小企業的規模/流動性溢價會人為提高內在價值。
4. **忽視周期性**：對於周期性業務使用峰值FCF利潤率。總是通過完整業務周期正常化FCF。
5. **忽視股票報酬**：SBC 是一個真實的稀釋成本。須從營運現金流中扣除。
6. **單一情境思維**：總是運行樂觀、基礎、悲觀情境。
7. **隱藏的營運資金和資本支出變動**：快速增長的公司經常消耗大量營運資金。
8. **貨幣和地域組合**：對於國際業務，需按地區進行預測，適用合適的折現率。

## DCF 與相對估值

**當DCF最可靠**：
- 穩定成熟企業，具有可預測、連續的FCF
- 輕資產，具有高FCF轉化率的企業（如軟體、消費品牌）
- 擁有10年以上FCF生成歷史的公司

**當 DCF 不可靠**：
- 初期成長公司，沒有正FCF
- 高度周期性企業需對FCF進行大幅度判斷的正常化
- 金融類公司（銀行、保險、REITs）—使用賬面市值、市盈率或股息折現模型
- 復甦情況下盈利能力不確定的公司
